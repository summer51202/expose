# Docs Index

## Folders

### `docs/design/`
- Product and feature design documents
- Architecture notes
- Mockups
- Implementation-oriented design specs

### `docs/runbooks/`
- Operational guides
- Deployment checklists
- Hosting and environment setup instructions

### `docs/issue-tracker/`
- Known issues
- Deployment pitfalls
- Troubleshooting records and postmortem-style notes

### `docs/security/`
- Security review output
- Risk tracking
- Remediation backlog

### `docs/superpowers/plans/`
- Execution plans and implementation plans
